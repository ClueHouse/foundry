# A Hole in One — Drilling

Single-screen industrial drilling website based on the approved visual mockup.

## Interaction

The five header controls open premium dropdown menus rather than full-screen overlays:

- Our Services
- About Us
- Projects
- Safety
- Get a Quote

The page is intentionally non-scrolling and scales as a fixed 3:2 canvas.

Open `index.html` directly or serve the folder with VS Code Live Server.
