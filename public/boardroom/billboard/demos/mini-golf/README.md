# A Hole in One — Tropical Mini Golf

A complete single-screen, no-scroll promotional website based on the approved Paradise Putt Putt visual direction, rebranded as **A Hole in One**.

## Run

Open `index.html` directly, or use VS Code Live Server.

## Interactive controls

Exactly five customer-facing controls:

1. The Course
2. Our Story
3. Locations
4. Parties & Events
5. Book Now

Each opens a distinct full-screen overlay with its own colour treatment. Press Escape, click the close button, or click outside the panel to close an overlay.

## Structure

- `index.html`
- `css/style.css`
- `js/app.js`
- `assets/images/puttputt-hero.webp`

The booking form is a front-end demonstration and is intentionally not connected to a backend service.
